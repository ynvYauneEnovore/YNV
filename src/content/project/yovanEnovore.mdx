# Habilidades en Desarrollo de Software

Como profesional de las Tecnologías de la Información, cuento con experiencia en diversas áreas del desarrollo de software. A continuación, detallo mis competencias en diferentes lenguajes de programación y maquetado:

## HTML (80%)

Como desarrollador de software, domino HTML en un 80%. He utilizado HTML como lenguaje fundamental en el desarrollo de páginas y aplicaciones web atractivas y funcionales. Implemento estándares de diseño web para garantizar la compatibilidad con múltiples navegadores y dispositivos, ofreciendo una experiencia de usuario excepcional.

## CSS (70%)

Mi experiencia en desarrollo web incluye un sólido dominio de CSS al 70%. Diseño y personalizo interfaces de usuario atractivas y responsivas para sitios web y aplicaciones, optimizando la presentación visual y la usabilidad de los proyectos. Mantengo una estructura de código limpio y eficiente, siguiendo las últimas tendencias de diseño y asegurando la compatibilidad con diversos navegadores y dispositivos.

## C# (50%)

En mi trayectoria, he desarrollado competencias en C# al 50%. Esto me ha permitido contribuir a proyectos de aplicaciones de escritorio y web. Estoy familiarizado con los conceptos fundamentales de C# y colaboro en la creación de componentes y módulos de software utilizando este lenguaje.

## C++ 

Mi experiencia en C++ me ha permitido participar en proyectos de desarrollo de software de nivel medio. Estoy comprometido en mejorar mis habilidades en este lenguaje para contribuir de manera más significativa a proyectos futuros.

## PHP (90%)

Como profesional en desarrollo web, tengo un sólido dominio del 90% en PHP. Esto me capacita para crear aplicaciones web dinámicas y funcionales de alta calidad. He trabajado con frameworks populares como Laravel y Symfony, lo que me permite crear soluciones web eficientes y escalables. Mantengo un código limpio y seguro, siguiendo las mejores prácticas de programación.

## SQL (80%)

Tengo un sólido dominio del 80% en SQL (Structured Query Language). Mi experiencia abarca desde el diseño y la optimización de bases de datos hasta la creación de consultas complejas para extraer y gestionar datos de manera eficiente.

## TypeScript (40%)

Cuento con un nivel de competencia del 40% en TypeScript. He participado en proyectos de desarrollo de aplicaciones web y front-end, y estoy comprometido en mejorar mis habilidades en este lenguaje.

## JavaScript (90%)

Con un sólido dominio del 90% en JavaScript, tengo una amplia experiencia en el desarrollo de aplicaciones web interactivas y dinámicas. He trabajado con bibliotecas y frameworks populares como Angular, React y Vue.js. Mi enfoque se centra en escribir código limpio y mantenible, siguiendo las mejores prácticas de programación y aplicando patrones de diseño sólidos.
